1. Установка
-Встановіть систему WordPress, створіть веб-сайт. Можна використовувати безкоштовний хостинг або свій.
Пропоновані хостинги:
--InfinityFree (https://infinityfree.net/)
--AwardSpace (https://www.awardspace.com/)
--000WebHost (https://www.000webhost.com/)
--Freehostia (https://www.freehostia.com/)
--ByetHost (https://byet.host/free-hosting)
--x10Hosting (https://x10hosting.com/)
--AccuWeb Hosting (https://www.accuwebhosting.com/web-hosting/free-wordpress-hosting)
--GoogieHost (https://googiehost.com/freehosting.html)
--Hostinger (https://www.hostinger.com/free-wordpress-hosting)
-Встановіть тему https://understrap.com/ (Тема використовує Bootstrap CSS Framework)
-Створіть дочірню тему для налаштування та доробки.
-Встановіть розширення ACF (Advanced Custom Fields) зі сторінки http://wordpress.org/plugins/advanced-custom-fields/
2. Створіть новий тип запису "Об'єкт нерухомості".
3. Використовуючи ACF, створіть набір полів для об'єктів нерухомості:
-"Назва будинку" (поле введення)
-"Координати місцезнаходження" (поле введення)
-"Кількість поверхів" (вибір зі списку 1-20)
-"Тип будівлі" (вибір зі списку панель/цегла/піноблок)
-"Зображення"
4. Заповніть базу даних інформацією про 7-10 будинків.
5. В шаблоні дочірньої теми головної сторінки додайте виведення Ajax фільтру і список будинків. Під блоком фільтра виводьте список будинків. Кожна позиція має зображення, заголовок та короткий опис з посиланням на сторінку будинку.
6. Ajax Фільтр по полям: "Кількість поверхів" та "Тип будівлі".
7. Створіть користувача-адміністратора та надайте йому доступ для перевірки.
8. Створіть публічний репозиторій на GitHub і завантажте туди код вашого веб-сайта.
---------------
P.s. Нам не важлива візуальна частина, тому використовуйте готові рішення за допомогою Bootstrap CSS
